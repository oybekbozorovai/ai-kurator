# 1-dars: Namuna kirish darsi (test uchun)

## Mavzu: Python o'zgaruvchilari

Python'da o'zgaruvchi yaratish juda oson. Siz shunchaki nom va qiymat yozasiz:

```python
ism = "Ali"
yosh = 25
boyi = 1.75
```

Python o'zgaruvchi turini avtomatik aniqlaydi. Bu — **dynamic typing** deb ataladi.

## O'zgaruvchi nomlash qoidalari

1. Faqat harf, raqam va `_` belgisidan foydalanish mumkin
2. Raqam bilan boshlanmasligi kerak
3. Probel ishlatib bo'lmaydi
4. O'zbek harflari texnik jihatdan ishlaydi, lekin tavsiya etilmaydi

**To'g'ri misollar:** `talaba_ismi`, `yosh`, `_yashirin`, `son1`
**Noto'g'ri misollar:** `1-talaba`, `talaba ismi`, `talaba!`

## Uy vazifasi

3 ta o'zgaruvchi yarating: ismingiz, yoshingiz va boyingiz. Keyin ularni `print()` orqali ekranga chiqaring.
